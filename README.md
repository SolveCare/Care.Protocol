# Care.Protocol

**Care.Protocol Specifications:** 

Care.Protocol defines the entities and the behavior of these entities which together forms the Care.Network. Care.Protocol not only specifies the rules for the network but also enforces the rules on the Network components to ensure it complies with pre-defined behavior. Every version of Care.Protocol is published to the Blockchain which helps in immutable, traceable and easily distributable Networks.

The Care.Protocol defines the set of rules or business contracts for the communication between components and governs the complete network.
